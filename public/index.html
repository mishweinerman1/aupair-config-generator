üìù How to Use:
  1. First time setup (admin only): Click "Admin Settings" and enter your GitHub token (see below for how to create one)
  2. Fill out the form with your Au Pair credentials and search preferences
  3. Click "Download Config File" - it will automatically save to GitHub
  4. The config will appear in the private repo: mishweinerman1/aupair-configs
  5. Download the config from GitHub and run the scraper on your computer
üîë Creating a GitHub Token (Admin Only - One Time):
  1. Go to: github.com/settings/tokens/new
  2. Set description: "AuPair Config Generator"
  3. Select scope: repo (full control of private repositories)
  4. Click "Generate token" at the bottom
  5. Copy the token (starts with ghp_) and paste it in Admin Settings above
  6. Important: Save this token somewhere safe - you can't see it again!
